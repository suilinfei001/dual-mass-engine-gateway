# 更新日志

本文件记录双引擎质量网关系统的所有重要变更。

## [1.0.0] - 2026-03-20

### 新增功能

#### 资源池管理模块
- **新模块**: 新增完整的资源池管理系统 (`src/modules/resource-pool/`)
  - 资源实例生命周期管理（创建、分配、释放、回收）
  - Testbed 管理及状态追踪和健康监控
  - 资源分类管理
  - 配额策略管理，支持灵活规则配置
  - 分配历史追踪和审计
  - Azure DevOps 部署流水线模板管理
  - 资源利用率监控仪表盘

#### 资源池调度器
- **核心调度逻辑**: 实现智能资源调度系统
  - 基于任务需求的自动资源匹配
  - 基于优先级的分配及配额强制执行
  - 资源健康监控和自动恢复
  - 可配置并发限制的任务处理

#### 部署集成
- **部署服务**: 完整集成 Azure DevOps
  - 产品部署工作流
  - 快照创建和回滚功能
  - 基于 SSH 的远程部署支持
  - 部署任务追踪和状态管理

#### AI 请求池
- **全局并发控制**: 设计 AI 请求池防止大模型过载
  - 可配置池大小（默认 50，最大 200）
  - 单事件并发限制（默认 20，最大 50）
  - 自动请求排队和等待

### 改进优化

#### 前端 UI/UX 优化
- **资源池页面**: 使用数据密集型仪表盘模式完整重设计
  - 渐变色头部，优化视觉层次
  - 响应式设计，适配所有屏幕尺寸（375px - 1440px）
  - 增强状态指示器和徽章
  - 改进表单布局，增加验证反馈
  - 环形图展示资源利用率
  - 11 个资源池页面统一设计语言

#### 事件接收器模块
- 重构 event-receiver 模块架构
- 新增 webhook 事件 ID 返回，便于追踪
- 增强事件状态管理，支持 `cancelled` 状态
